Guide d'utilisation

Tout ce qu'il faut savoir pour démarrer avec Bilan Pro

1Démarrer la conversation

Cliquez sur le bouton "Démarrer gratuitement" depuis la page d'accueil. Vous êtes redirigé vers WhatsApp avec un message pré-rempli — il vous suffit d'appuyer sur envoyer.

2Créer votre PIN

Bilan Pro vous accueille et vous demande de créer un code PIN à 4 chiffres, puis de le confirmer. Ce code protège votre compte.

3Choisir une question secrète

Sélectionnez une question secrète parmi 3 options et répondez-y. Cela vous permettra de récupérer votre PIN si vous l'oubliez.

4Message de bienvenue

Une fois le compte créé, vous recevez ce message de bienvenue suivi d'un tutoriel guidé en 4 étapes :

Compte cree avec succes ! Bienvenue sur Bilan Pro, ton assistant comptable sur WhatsApp ! 14 jours d'essai gratuit. Profites-en pour tout tester ! Apres, un abonnement mensuel sera propose selon ton pays. On va faire un essai rapide ensemble. ETAPE 1/4 — Tu fais une vente ? Tape exactement : vt 1000 test

5Tutoriel guidé

Bilan Pro vous guide à travers 4 exemples pratiques — vente, dépense, crédit et bilan du jour. Suivez chaque étape pour apprendre les commandes de base.

6Enregistrer une transaction

Utilisez les abréviations suivies du montant et d'une description. Bilan Pro enregistre instantanément et confirme par message.

vt = Vente (argent reçu immédiatement)
dp = Dépense
cr = Crédit client (argent à recevoir)
rb = Remboursement (client qui paie sa dette)

7Consulter votre bilan

Tapez blj pour le bilan du jour, blm pour le mois, blt pour le trimestre, bls pour le semestre, bla pour l'année.

8Accéder au panel client

Rendez-vous sur www.bilanpro.app/client pour vous connecter avec votre numéro et votre PIN. Accédez à vos statistiques, l'historique complet et exportez en Excel ou PDF.
